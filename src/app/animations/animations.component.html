Animationen

CSS-Animationen

Mithilfe von @keyframes können Animationen im CSS erstellt werden.

{{ ".subject {" }}
animation: pulse 2000ms infinite;
{{ "}" }}

{{ "@keyframe pulse {" }}
{{ "0% {"}}
transform: scale(1);
{{ "}"}}
{{ "25% {"}}
transform: scale(2);
background: #FC9038;
{{ "}"}}
{{ "100% {"}}
transform: scale(1);
{{ "}"}}
{{ "}" }}

Animated GIF

Eine Animation kann auch als GIF direkt ins HTML eingebunden werden.

<img class="gif" src="travis-scott.gif" alt="GIF">

SVG-Animationen

Es können Animationen mithilfe von SVG erstellt werden. Auf die Elemente eines SVGs kann direkt vom CSS zugegriffen werden, wie auf alle andere HTML-Tags.

<svg class="svg"></svg>

{{ ".svg {" }}
animation: blink 2000ms infinite;
{{ "}" }}

{{ "@keyframe blink {" }}
{{ "0% {"}}
opacity: 0;
{{ "}"}}
{{ "25% {"}}
opacity: 1;
{{ "}"}}
{{ "100% {"}}
opacity: 0;
{{ "}"}}
{{ "}" }}

Scroll-Animationen

Scroll-Animationen sind Animationen, welche durch das Scrollen ausgelöst werden. Mithilfe von Javascript können Scroll-Events im Fenster abgefangen werden. Danach kann je nach Zustand vom Scrollen der Animation einen bestimmten Zustand geben.

Beispiel:
Webseite nicht gescrollt: Animation zu 0% sichtbar.
Webseite zur Hälfte gescrollt: Animation zu 50% sichtbar.
Webseite ganz nach unten gescrollt: Animation zu 100% sichtbar.
